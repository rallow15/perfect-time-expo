# Perfect Time ⏱

Jeu de party — estimez le temps invisible !

## Installation

### 1. Prérequis
- [Node.js](https://nodejs.org) (version LTS)
- [VS Code](https://code.visualstudio.com)
- Application **Expo Go** sur ton iPhone (App Store)

### 2. Ouvrir le projet dans VS Code
Ouvre le dossier `perfect-time` dans VS Code.

### 3. Terminal VS Code (`Ctrl+``)
```bash
npm install
npx expo start
```

### 4. Lancer sur ton téléphone
- Un QR code apparaît dans le terminal
- Ouvre l'app **Expo Go** sur ton iPhone
- Scanne le QR code → le jeu se lance !
